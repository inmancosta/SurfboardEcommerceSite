# SurfboardEcommerceSite

Welcome to Flowmazon! This is a sophisticated e-commerce platform specifically tailored for surfboard enthusiasts. This application was developed using TypeScript, React, HTML/CSS, and MongoDB as its database solution.

<h2>Features</h2>

<h3>Client-Side Components</h3>
The client-side of this application is built using TypeScript and React, ensuring a seamless and intuitive user experience. Here are some key features:

<b>Google Authentication</b>: Users can securely log in using their Google accounts, ensuring a streamlined and secure authentication process. 
<b>Product Catalog</b>: A dynamic and visually appealing interface displaying a wide range of surfboards available for purchase.
<b>Shopping Cart</b>: Users can easily add desired surfboards to their carts, manage quantities, and proceed to checkout effortlessly.
Server-Side Components
The server-side components of this application are robust and scalable, providing a foundation for secure data management and user interactions. Key features include:

<b>MongoDB Database Integration</b>: Utilizing MongoDB as the primary database solution for efficient and flexible data storage.
<b>Seller Capabilities</b>: Users can post their own surfboard listings, enabling them to become sellers on the platform.
Getting Started

To run this application locally, follow these steps:

Clone Repository: Clone this repository to your local machine.
Install Dependencies: Use npm or yarn to install all necessary dependencies.

Run the Application: Start the application using "npm run dev"

<h2>Technology used</h2>

TypeScript: Ensuring type safety and enhanced code quality.
React: Providing a dynamic and responsive user interface.
HTML/CSS: Structuring and styling the application for an engaging user experience.
MongoDB: As the robust and flexible database solution.
Google Authentication API: Enabling secure and convenient user login functionality.
